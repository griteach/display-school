# Spike Prime 익히기

> 우연한 기회에 접하게 된 Lego Spike Prime! 기본적인 내용부터 파악해보자!

# 소프트웨어 사용의 기초

- 스파이크 프라임 앱 켜기
- 스파이크 프라임 `시작` 선택하기
- 6가지 단계 연습하기
  - 모터연습
    - A~F 포트를 잘 확인할 것!
    - 각 센서, 모터마다 고유 아이콘이 있어서 코딩할 때 쉽게 확인할 수 있음.
  - 컬러센서 연습
    - 모니터 상단에서 센서의 값을 실시간으로 확인할 수 있음.
  - 거리센서 연습
    - cm, inch, % 단위 잘 확인
    - 모니터 상단에서 센서의 값을 실시간으로 확인할 수 있음.
  - 힘 센서 연습
    - 누름, 누르지 않음, 강하게 누름, 압력이 변경됨.
    - 모니터 상단에서 센서의 값을 실시간으로 확인할 수 있음.
  - 자이로 센서 연습
    - 앞, 뒤, 좌, 우, 가만히 있는 상태, 어디든 움직이는 상태

# 코뿔소 활동하기

---

![코뿔소 코드](./rhino.png)

1. 전진 및 정지하기
   1. 프로그램이 시작될 때 블록 사용
   2. 반복하기 사용
      1. 반복 연습하기
      2. 횟수 반복, 무한 반복
   3. 조건문 사용하기
      1. 조건문 연습
      2. 제어, 연산 연습하기
   4. 프로그램 시작될 때 -> 모터 세팅 -> 무한반복 -> 주행시작 -> 만약 조건(힘 센서 누름이면) 동작 멈추기 -> 3초 대기하기 -> 다시 동작 속도 50% 정하기
2. 거리 센서에 따라 후진하기
   1. 거리 센서 조립하기
   2. 포트 D에 연결하기
   3. 만약 조건(거리센서 8cm 이하면) 후진하기
      1. 랜덤으로 방향 조절하기
   4. 다시 동작 속도 50% 정하기
3. 컬러 센서로 부스터 명령넣기
   1. 컬러 센서 조립하기
   2. 색상 정하기
   3. 만약 조건(빨강)이면
      1. 소리 재생하기(부스터에 맞는 소리...?)
      2. 소리 녹음도 가능함.
      3. 소리 재생 후 동작 속도 100%로 변경하기
      4. 3초 대기하기
      5. 동작속도 50%로 변경하기
4. 게임만들기
5. 코로 찌르면 점수를 얻습니다!
6. 다같이 벽을 향해 돌진!
7. 코로 한 번 찌를 때 마다 속도가 향상됩니다.
8. 변수 생성하기 (점수, 속도)
9. 동작 속도를 변수 속도로 정하기
10. 코가 눌리면 속도 추가 및 점수 추가
11. 디버깅 가능한 부분
12. 점수 10점까지?
13. 부딪칠때마다 10씩 늘리는 것이 아니라 랜덤으로 부여하기
14. 애초에 나의 속도는 랜덤으로 부여?
15. 대기시간 조절?
